## How to push to remote GitHub repo named `iot`
1. Create a GitHub repository named `iot` on your GitHub account.
2. Add the remote and push:

For HTTPS:
```
git remote add origin https://github.com/<USERNAME>/iot.git
git branch -M main
git push -u origin main
```

For SSH:
```
git remote add origin git@github.com:<USERNAME>/iot.git
git branch -M main
git push -u origin main
```

Replace `<USERNAME>` with your GitHub username.
